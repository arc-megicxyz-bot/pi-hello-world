# pi-hello-world

Hello World project scaffolded to be evolved using **Pi coding agent**.

## Requirements

- Node.js >= 20

## Install

```bash
npm install
```

## Run

Dev (TypeScript, no build):

```bash
npm run dev
```

Build:

```bash
npm run build
```

Run built output:

```bash
npm start
```

## Test

```bash
npm test
```

## Lint/Format/Check

```bash
npm run check
```

## Use with Pi

```bash
npm i -g @mariozechner/pi-coding-agent
pi
```

Then ask Pi to:
- add CLI args
- add more tests
- publish as an npm package
